# 🍔 Food Delivery SQL Analytics Project

## Overview
End-to-end SQL analytics project on a Swiggy/Zomato-style
food delivery platform using MySQL.

## Database Schema
- customers — customer profiles and city data
- restaurants — restaurant details and cuisine types
- orders — order transactions with status and amount
- delivery_agents — agent details and ratings
- order_delivery — pickup and drop time per delivery

## Topics Covered
- Joins & Aggregations (revenue, cancellation rate)
- Window Functions (RANK, DENSE_RANK, LAG, ROW_NUMBER)
- Subqueries & CTEs (customer segmentation, agent performance)
- Performance Optimization (indexes, EXPLAIN, EXISTS vs IN)
